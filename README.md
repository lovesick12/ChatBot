# TrackBot – Internship Assistant Chatbot

TrackBot is an AI-powered chatbot designed to assist students during their internship. It provides instant answers to questions related to tasks, submissions, schedules, mentors, deadlines, and other internship information.

## Features
- Answer internship-related queries
- Task and submission guidance
- Schedule and deadline information
- Mentor contact details
- User-friendly chat interface

## Technologies Used
- Python
- Flask/Django
- HTML, CSS, JavaScript
- SQLite/MySQL
- AI/NLP APIs (optional)

## Future Enhancements
- Voice support
- Multi-language support
- Integration with LMS platforms
- Personalized notifications
